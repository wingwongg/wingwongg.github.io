What is Reclaimr?

Reclaimr is a screen time management app that helps you set daily limits on selected apps. When your screen time runs out, Reclaimr restricts access automatically.

Earn extra screen time by completing productive tasks like reading, exercise, or journaling — all tracked inside the app.

Reclaimr empowers you to build healthier digital habits and make screen time more intentional.

Our Mission

Reclaimr is built to help individuals develop healthier screen habits. By setting daily limits and rewarding real-world productivity, our app encourages focus, balance, and self-awareness in digital life. We're committed to supporting families and promoting digital wellbeing.

Key Features

  • Set daily screen time limits across selected apps
  • Earn screen time by completing real-life productivity goals
  • iOS Family Control & Screen Time API integration
  • Simple, focused design for self-discipline
